# Skooldio-homework-backend

Command-line based script using Node.js for function run-length encoding

## Installation

```bash
npm install
```

## Build

```bash
$ npm run build
```

## Usage

```bash
$ npm run encode <string to encode>

# example
# $ npm run encode school
# $ s1c1h1o2l1
```

## Test

```bash
$ npm test
```
